# Fehlende Implementierungen

## Übersicht

Aufgrund von einem Zeitlimit von ca. 1h und die begrenzte Größe des Projektes wurde auf folgende Implementierungen verzichtet.

## Fehlende Implementierungen (keine Features)

- Fehlende Tests im Task-Module. Die E2E, Integration und Unit-Tests im User-Module wurden in Zusammenarbeit mit ClaudeCode erstellt :)
- Fehlende Checks und Custom Error Messages im Task-Module
- Fehlende/Mangehalfte Code Dokumentation
- Fehlende Dokumentation der Endpunkte mit Swagger
- Monolithische Architektur aufgrund von Zeit und größe der Applikation
- Fehlende Validation der Inputs durch z.B. Zod
- Fehlendes Dockerfile für die NestJS Anwendung
- Fehlendes Logging
- Globales Error Handling
- Pagination
- HealthChecks
- CI/CD Pipeline
